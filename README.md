## Features

- 🎯 **150+ Interactive Lessons** - From basics to advanced
- ⚡ **Real-Time Execution** - Run TypeScript in your browser
- 🌐 **Bilingual** - English & Portuguese
- 📱 **Mobile-First** - Learn on any device
- 🔒 **GitHub OAuth** - Secure authentication
